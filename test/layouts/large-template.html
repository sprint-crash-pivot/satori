A Quick Tip

You may want to create a separate SASS stylesheet and include Satori's classes to simplify your code. The following code is recommended:

section {
	@extend bg-white;

	footer {
		@extend bg-grey;
	}
}

This allows you to avoid having to include bg-white and bg-grey in each section. It leads to slightly more maintainable code.
