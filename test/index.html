Usage

Install the sass and listener gems using the following commands:

gem install sass

gem install listener

Remember to run the following command from the root directory of the project:

sass --watch test/styles:tmp

In order to update the gem, you'll need to increment the version number in /lib/satori/version.rb and run the following command after pushing changes to the GitHub repository:

rake release
